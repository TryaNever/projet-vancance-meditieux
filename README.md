# Projet Vacance Méditieux

## Description
Ce projet a été créé dans le cadre d'une évaluation notée en première année. Il s'agit d'une application conçue avec **JavaScript, HTML et SCSS**.

## Collaborateurs
Ce projet a été réalisé en collaboration avec :
- [Augustin](https://github.com/Augustin51)

## Technologies utilisées
- **HTML** : Structure du projet
- **SCSS** : Mise en page et styles
- **JavaScript** : Logique et interactivité

## Installation
1. Clonez le dépôt :
   ```bash
   git clone <lien-du-repo>
   ```
2. Accédez au dossier du projet :
   ```bash
   cd projet-vacance-meditieux
   ```
3. Ouvrez le fichier `index.html` dans un navigateur.

## Fonctionnalités
- Interface dynamique et responsive
- Expérience utilisateur optimisée avec SCSS
- Interactions et animations en JavaScript
